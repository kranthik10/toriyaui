# ToriUI

ToriUI is a shadcn-style UI component system for Expo and React Native.

## Goals
- Copy-in components: users own their UI code.
- Small runtime surface area.
- Composable primitives and predictable theming.

## Quick start (local)
- Run the CLI in this repo: `npm run cli -- init`
- Add a component: `npm run cli -- add button`

Templates live in `templates/` and are copied into your app under `components/ui/`.

## Starter components
`button`, `input`, `card`, `badge`, `avatar`, `tabs`
